# C++ File Organizer

A lightweight, high-performance desktop utility that automatically organizes files by grouping items with matching names into unique, dedicated subfolders. 

## 🚀 Features
- **Smart Grouping**: Groups files sharing the same name (ignoring extensions, e.g., `photo.jpg` and `photo.png`) into a single subfolder.
- **Automated Folder Creation**: Automatically generates required subfolders on the fly.
- **Fast & Lightweight**: Runs instantly inside a command terminal with zero bloat or background background processes.
- **100% Standalone**: No installation required. Just run the `.exe` and go.

## 📦 How to Download and Run

1. Download the latest `FileOrganizer.exe` from the releases section.
2. Double-click `FileOrganizer.exe` to open the program interface.
3. Provide the application with two paths when prompted:
   - **Source Folder Path**: The folder containing your disorganized files.
   - **Destination Folder Path**: The folder where you want your new subfolders to be created.
4. Press **Enter** and watch your files organize instantly!

## ⚠️ Important Windows Usage Tips

### 1. Handling Windows SmartScreen Warnings
Because this executable is an independent, un-signed utility, Windows SmartScreen may show a popup saying *"Windows protected your PC"* when you first run it.
- **To bypass this:** Click **"More info"** on the popup, then click the **"Run anyway"** button. The application is completely safe and only modifies the files inside the specific folders you provide.

### 2. Copying Paths Easily
Instead of typing out long file paths manually, you can easily copy folder locations in Windows:
- **Windows 11**: Right-click your target folder and select **"Copy as path"**.
- **Windows 10**: Hold the `Shift` key, right-click your target folder, and select **"Copy as path"**.
- *Note: If the copied path includes quotation marks (e.g., `"C:\My Folder"`), make sure to remove them when pasting into the terminal.*
